### 🛠️ [DkvsRaft](https://github.com/Amrit-mishra07/DkvsRaft)

A from-scratch **Raft consensus implementation** in Go. Built leader election with randomized timeouts (150–300ms failover), log replication, quorum-gated commits, and linearizable reads over gRPC/Protobufs across a 3-node Docker Compose cluster. Includes a chaos engineering test suite that kills the leader mid-write and verifies sub-300ms re-election and log truncation on recovery.

### 🛡️ [SentinelAI](https://github.com/Amrit-mishra07/SentinelAI)

A production-grade **AI-powered SAST platform** built as a monorepo. Orchestrates Bandit, Semgrep, and ESLint concurrently via async Celery workers backed by Redis, uses OpenAI GPT-4 to auto-generate security patches, and serves results through a Next.js 14 dashboard. Full Docker Compose stack with Nginx reverse proxy, PostgreSQL, JWT auth, and 28 passing tests.

### 🤖 [ADK 2.0 Ambient Agent](https://github.com/Amrit-mishra07/ADK_2.0_Ambient_agent)

An event-driven **FastAPI microservice** using Google ADK 2.0. Triggered by Pub/Sub events, it autonomously processes expense reports using LLM-based PII redaction, prompt injection detection, and Human-in-the-Loop escalation for high-value requests. Includes a synthetic trace evaluation pipeline for continuous quality monitoring.
